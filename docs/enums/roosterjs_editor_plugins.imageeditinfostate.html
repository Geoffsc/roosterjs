Enumeration ImageEditInfoState

State of an edit info object for image editing. It is returned by checkEditInfoState() function

FullyChanged

FullyChanged:
  • Defined in roosterjs-editor-plugins/lib/plugins/ImageEdit/editInfoUtils/checkEditInfoState.ts:44

When this value is returned, it means the image is edited by either cropping or rotation, or both. Image source can't be reused, need to generate a new image src data uri.

ResizeOnly

ResizeOnly:
  • Defined in roosterjs-editor-plugins/lib/plugins/ImageEdit/editInfoUtils/checkEditInfoState.ts:29

The edit info shows that it is only potentially edited by resizing action. Image is not rotated or cropped, or event not changed at all.

SameWithLast

SameWithLast:
  • Defined in roosterjs-editor-plugins/lib/plugins/ImageEdit/editInfoUtils/checkEditInfoState.ts:37

When compare with another edit info, this value can be returned when both current edit info and the other one are not been rotated, and they have same cropping percentages. So that they can share the same image src, only width and height need to be adjusted.
